# π-pugger

A minimal, single-file web app for memorizing the digits of pi — 10 at a time.

## Features

- **Learn mode** — see the digits, read and memorize
- **Practice mode** — type from memory with instant per-digit validation
- **Range selector** — pick exactly which digits to drill (1–10, 11–20, etc.)
- **10-digit milestones** — visual markers at every 10th digit boundary
- **Progress tracking** — streak, daily count, and longest memorized (stored in localStorage)
- **Norwegian / English** — follows your system language, with a manual toggle

## Usage

Open `index.html` in any browser. No server, no dependencies, no build step.
